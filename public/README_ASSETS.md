# Assets à ajouter

## Logo du Ministère
Placez le logo officiel du Ministère de la Jeunesse dans :
```
public/logo/logo-ministere.png
```

**Format recommandé** : PNG avec fond transparent, dimensions optimales : 200x200px minimum

---

## Images du Carousel Hero
Placez les 3 images pour le carousel hero dans :
```
public/images/hero/hero-1.jpg
public/images/hero/hero-2.jpg
public/images/hero/hero-3.jpg
```

**Format recommandé** : JPG ou WebP, dimensions optimales : 1920x1080px (16:9)
- **hero-1.jpg** : Image principale pour le premier slide
- **hero-2.jpg** : Image pour le deuxième slide (Éveil Patriotique)
- **hero-3.jpg** : Image pour le troisième slide (Opportunités)

---

## Logos des Partenaires (optionnel)
Si vous avez les logos des partenaires, placez-les dans :
```
public/logo/partners/unesco.png
public/logo/partners/unicef.png
public/logo/partners/cisco.png
public/logo/partners/bad.png
public/logo/partners/pnud.png
public/logo/partners/ua.png
```

**Format recommandé** : PNG avec fond transparent ou blanc, dimensions : 150x80px environ

---

**Note** : Si les images ne sont pas trouvées, le site affichera des placeholders automatiques.
